# semaforo-inteligente.
- Nome do projeto: opsemafros
- Nome do desenvolvedor: Pestana
- Objetivo do projeto: Diminuir o transito e reduzir a taxa de acidentes
- Linguagem utilizada: C#
- Descrição resumida do funcionamento: O semafaro funcionara com cameras e um jogo de pausa entre a troca de passagem de via, e um tempo extra para a via mais congestionada.
- Como executar o programa, quando houver código: Executar o projeto dentro do Visual Code utilizando o GCC.
